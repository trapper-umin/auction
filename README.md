## **Technology stack**
 - BACKEND: JAVA CORE, SPRING BOOT 3, SPRING DATA JPA, SPRING SECURITY, SPRING REST, HIBERNATE, MAVEN, POSTGRESQL
  
- FRONTEND: JS, REACT
  
- Principles: OOP, SOLID, REST
